Let $x = [x_1, x_2, \ldots, x_n]^T$ and $y = [y_1, y_2, \ldots, y_n]^T$.

The design matrix $X$ contains two columns $$X = \begin{bmatrix} 1& x_1 \\ 1& x_2 \\ \vdots& \vdots \\ 1& x_n \end{bmatrix} $$ We can calculate the regression coefficients $M = [\theta_0, \theta_1]^T$ by $$M = (X^T X)^{-1} X^T y$$

# Design matrix X = [1 x]
X = np.concatenate((np.ones((10,1)), x.reshape(10,1)), axis=1)

# Target values y
Y = y.reshape((10,1))

# Calculate the regression coefficients
M = la.inv(np.dot(X.T, X)).dot(X.T).dot(Y)

z = X.dot(M)
plt.plot(x, z)
plt.plot(x, y, 'o')
plt.show()

2. Regression, one variable, fitting a second order polynomial

We fit a second order model $$y_i = \theta_0 + \theta_1 x_i + \theta_2 x_i^2$$ to the data. The design matrix $X$ now contains three columns $$X = \begin{bmatrix} 1& x_1& x_1^2 \\ 1& x_2& x_2^2 \\ \vdots& \vdots& \vdots \\ 1& x_n& x_n^2 \end{bmatrix} $$

3. Fitting a higher order polynomial

We the use the Python command polyfit to fit a higher order polynomial $$y_i = \theta_0 + \theta_1 x_i + \theta_2 x_i^2 + \cdots + \theta_k x_i^k$$ to the data.

A 9th order polynomial has 10 parameters and can be made to fit the 10 datapoints perfectly. However, the polynomial often oscillates wildly between datapoints. It is therefore bad for estimating new values. This is called overfitting: the model learns the data very well, but is bad at predicting previously unseen values.

Try yourself, how model behaves if you change the order $k$ of the polynomial in the command "polyfit(x, y, k)".

4. Regression, multiple variables

We can use normal equations to fit a model, which explains a single target variable $y$ by several features $x^{(1)}, x^{(2)}, \ldots, x^{(m)}$. The model is $$y_i = \theta_0 + \theta_1 x^{(1)}_i + \theta_2 x^{(2)}_i + \cdots + \theta_m x^{(m)}_i$$

We create a design matrix $$X = \begin{bmatrix} 1& x_1^{(1)}& x_1^{(2)}& \ldots& x_1^{(m)} \\ 1& x_2^{(1)}& x_2^{(2)}& \ldots& x_2^{(m)} \\ 1& x_3^{(1)}& x_3^{(2)}& \ldots& x_3^{(m)} \\ \vdots& \vdots& \vdots&\ddots &\vdots \\ 1& x_n^{(1)}& x_n^{(2)}& \ldots& x_n^{(m)} \end{bmatrix} $$

The coefficients $M = [\theta_o \theta_1 \theta_2 \cdots \theta_m]^T$ can be calculated as above $$M = (X^T X)^{-1} X^T y$$

from keras.datasets import boston_housing

# Download the data

(train_data, train_targets), (test_data, test_targets) = boston_housing.load_data()


# Create the feature matrices

n = len(train_data)  # 404
m = len(test_data)  # 102

X = np.concatenate((np.ones((n, 1)), train_data), axis=1)
X_test = np.concatenate((np.ones((m, 1)), test_data), axis=1)


# Create target vectors

y = train_targets.reshape(n, 1)
y_test = test_targets.reshape(m, 1)

# Calculate the regression coefficients

M = np.linalg.inv(X.T.dot(X)).dot(X.T).dot(y)

y_estimate = X_test.dot(M)
results = np.concatenate((y_test, y_estimate, y_estimate - y_test), axis=1)
