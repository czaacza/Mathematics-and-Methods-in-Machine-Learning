Metrics¶

Mean Absolute Error (MAE) is mean of the absolute values of errors. It takes the error between the real and predicted value and calculates the mean. $$MAE = \frac 1n\sum_{i=1}^n|y_i-\hat{y}_i|$$ It won't punish large errors, so it's not used much.

Mean Squared Error (MSE) is the mean of squared errors. Meaning that it takes the square of the error and then calculates the mean. $$MSE = \frac{\sum_{i=1}^n(y_i-\hat{y}_i)^2}{n}$$

Root Mean Squared Error (RMSE) $$RMSE = \sqrt{\frac 1n\sum_{i=1}^n(y_i-\hat{y}_i)^2}$$

The R2s value is low which means that correlation is low. It seems to imply that the model doesn't fit the observed data well. However the predictions seem to work well with the testing data.
